Abrir e fechar janelas

Uma janela é automaticamente criada quando inicia a sua sessão no browser. Pode-se abrir outra janela através do menu File opção New Window. Para fechar a janela usa-se a opção Close ou Exit.

Estas operações também podem ser implementadas usando o JavaScript.

Abrir uma janela

Para abrir uma janela usa-se o método open. A expressão seguinte cria uma janela chamada msgWindow e que apresenta a página HTML: news.html:

msgWindow = window.open("news.html");

Correctamente seria:

<script language="JavaScript">
<!--
msgWindow = window.open("news.html");
//-->
</script>

Para abrir a página principal do TerràVista:

outraJanela = window.open("http://www.terravista.pt");

As janelas podem ter dois nomes. A expressão seguinte cria uma janela com dois nomes. O primeiro, msgWindow, usa-se no acesso a métodos e propriedades do objecto window; o segundo, displayWindow, usa-se nas directivas HTML, como forms, target ou com um link.

msgWindow = window.open("news.html", "displayWindow");

Quando se abre a janela pode-se especificar alguns altributos como dimensão, toolbar, scrollbars. Para uma janela sem toolbar e com scrollbars:

msgWindow=window.open(
       "news.html","displayWindow","toolbar=no,scrollbars=yes");

Brevemente estará aqui disponível toda a informação sobre os atributos.

Fechar uma janela

Para fechar uma janela usa-se o método close do objecto window. As expressões seguintes fecham a janela activa:

window.close();

self.close();

Para fechar a janela msgWindow, criada anteriormente usa-se:

msgWindow.close();
