Instruções de instalação para Jekyll

    No windows:
  • Instalar o chocolatey com o comando no Powershell com permissões de Administrador: "iwr https://chocolatey.org/install.ps1 -UseBasicParsing | iex". Se der SecurityError executar o comando no Powershell Set-ExecutionPolicy e inserir Unrestricted. Mais detalhes aqui
  • In the powershell command window install Ruby via Chocolatey: choco install ruby -y
  • Reopen a command prompt and install Jekyll: "gem install jekyll". If gives an SSL error check this page https://labs.sverrirs.com/jekyll/2-jekyll-gem.html
