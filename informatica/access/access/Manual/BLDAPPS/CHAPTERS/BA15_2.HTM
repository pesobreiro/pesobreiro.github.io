Setting Startup Options

You can set options in the Startup dialog box that determine how your application looks and behaves when it starts. For example, you can:

&nbspSpecify which form to display when your application opens.

&nbspIndicate which menus and toolbars can be used in your application.

&nbspSet the text that appears in the title bar while your application is running.

&nbspKeep the Database window from displaying.

&nbspPrevent the use of special keystrokes used to display the Database window (F11 or ALT+F1), show the Debug window (CTRL+G), or pause execution (CTRL+F11).

To display the Startup dialog box, click Startup on the Tools menu.

See Also   For more information on using the Startup dialog box, see "Setting Startup Options" in Chapter 1, "Creating an Application."
