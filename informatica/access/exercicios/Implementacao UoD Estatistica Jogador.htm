Definição da uma Base de Dados

 

Tarefas

 

 

A partir do Modelo Entidade Relação (ER) definido no exercício anterior, crie uma base de dados realizando o mapeamento das entidades, atributos e relações do modelo ER para tabelas, campos e relações da Base de Dados, considerando os seguintes aspectos:

 

-         A Base de Dados deverá ter definida os tipos de dados para cada campo, bem como as tabelas e relações. Na estruturação das tabelas e necessário considerar os campos/tabelas que são necessários para a implementação das relações.

-         As relações deverão considerar a implementação da integridade referencial como um mecanismo de validação e manutenção da coerência dos dados inseridos.

-         A base de dados deverá ter o nome de estatística atleta e às tabelas/campos serão atribuídas as mesmas designações que foram definidas no modelo ER.

 

Por último deverá ser feita a introdução de alguns dados de forma a verificar o funcionamento do modelo implementado.
