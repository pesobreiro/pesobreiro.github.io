Campo SET
®Atribui um valor (quer texto, quer um número) a um marcador. Para imprimir o valor em cada documento intercalado resultante, terá de inserir um campo de marcador no documento principal. O mesmo valor será repetido em cada documento intercalado. O marcador representa esse valor, até que lhe atribua um valo diferente.
®{ SET Telefone { IF { MERGEFIELD Distrito } = "LX" "1239782" "1239792" } }
®Lembre-se de nos contactar até 30 de Novembro através do telefone { Telefone }
