Tema do Trabalho

O trabalho pode ser feito a partir de dois temas à escola, nomeadamente:

  • Estatistica de Jogos.
  • Controle de Contactos.

    Com primeiro tema de trabalho pretende-se criar um repositório de dados onde existam vários indicadores que possam ser utilizados para uma análise pós-jogo com a flexibilidade fornecida pelas consultas, mais detalhe poderão ser encontrados aqui.

    No segundo tema de trabalho o objectivo é criar um repositório de dados que flexibilize o processo de envio de mailing, ou seja é criada uma base de dados para armazenar contactos que irá permitir a geração da impressão em série para o Word.
