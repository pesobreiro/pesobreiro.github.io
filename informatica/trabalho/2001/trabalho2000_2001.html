Tema do Trabalho

O trabalho pode ser feito a partir de dois temas à escola, nomeadamente:

  • Historial de Jogos.
  • Controle de Contactos.

    Com primeiro tema de trabalho pretende-se criar um repositório de dados onde existam vários indicadores que possam ser utilizados para uma análise pós-jogo com a flexibilidade fornecida pelas consultas. Esta Base de Dados permite armazenar o historial por jogo de um jogador através de alguns indicadores que foram considerados relavantes.
    Mais detalhes relativamente ao trabalho podem ser encontrados  aqui.

    No segundo tema de trabalho o objectivo é criar um repositório de dados que flexibilize o processo de envio de mailings.  É criada uma base de dados para armazenar contactos que irá permitir a geração da impressão em série para o Word, possibilitando ainda o armazenamento dos documentos enviados aos contactos e o controle de que documentos foram enviados a cada contacto.
