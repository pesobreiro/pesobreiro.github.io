Trabalho de Informática 2002/2003

O trabalho a ser desenvolvido baseia-se na implementação de uma base de dados que é constituída por diversos componentes e que deverão ser implementados, para o  seu desenvolvimento deverá ser feita uma planificação prévia que será utilizada para controlar a execução do projecto/base de dados.

A base de dados a desenvolver deverá ser baseada num problema concreto que se pretende resolver

O trabalho deverá ser entregue com:

Relatório 
Diagrama de Gantt  
Base de Dados 

O Trabalho deverá ser enviado por email para sobreiro@esdrm.pt até às 24:00 horas do dia 14 de Janeiro de 2003, o titulo da mensagem deverá estar identificado com Trabalho de Informática e no corpo da mensagem será colocado os elementos que constituem o grupo(número, nome e curso), bem como toda a documentação solicitada. 

 

Relatório

No relatório será colocado todos os elementos que incluem a modelação do problema até a sua implementação, bem como dificuldades encontradas no seu desenvolvimento. 

O relatório deverá entregue em suporte físico(papel) no dia da frequência, paginado e respeitando elementos: 

Margens
Margem Superior: 3 cm
Margem Inferior: 2 cm
Margem Esquerda: 3 cm
Margem Direita: 2 cm
Tipo de Letra
Arial
Tamanho 12
Espaçamento entre parágrafos 1,5

O relatório deverá incluir:

Capa, com identificação dos autores, título do trabalho, ano e local
Contra capa, com nomes dos autores, título do trabalho, objectivo e instituição a que se destina
Sumário, principais divisões do trabalho com a indicação das páginas
Lista de tabelas e figuras (caso o relatório contenha)
Núcleo do Trabalho:
Introdução
Desenvolvimento
Conclusão
Apêndices e anexos ( se necessários)
Bibliografia
Capa Final

Diagrama de Gantt(Planificação do projecto a desenvolver)

Antes de se realizar a implementação da Base de Dados deverá ser desenvolvido um projecto que permita identificar as tarefas e recursos utilizados pela Base de Dados, nomeadamente: 

Identificação das tarefas e recursos que envolvem o projecto
Atribuição de tempos
Estimação dos tempos envolvidos na execução do projecto

No diagrama deverão estar identificadas que tarefas foram executadas por cada elemento do grupo.

Base de Dados(Implementação da Base de Dados)

Definição do Universo do Discurso  (UoD).
Elaboração do Modelo da Base de Dados identificando as entidades, atributos e relações, utilizando o modelo entidade relação.
Aplicação da 1º, 2º e 3º Fórmula Normal.
Implementação da Base de Dados:
Definição das tabelas
Campos e selecção dos tipos de dados apropriados.
Atribuição da chave primária
Definição das relações e imposição ou não das regras de integridade referencial
Implementação de Interfaces, Consultas e Relatórios

A Base de Dados deverá ser desenvolvida utilizando o Microsoft Access e deverá ser entregue num formato compatível com o Access 97 ou 2000.

Qualquer dúvida ou esclarecimento em relação ao trabalho poderá ser colocado durante as aulas práticas, no horário de atendimento aos alunos ou por email para sobreiro@esdrm.pt
