Trabalho Prático de Informática

Condição Física

 

O trabalho deve ser entregue no dia do teste de informática.

Deve ser entregue em disquete num envelope devidamente identificado com os autores do trabalho (Curso, Número e Nome)

Enunciado do Trabalho:

Um treinador pretende criar uma Base de Dados de forma a controlar a performance dos seus atletas nos jogos de futebol, para pretende criar uma base de dados que lhe permita obter a informação que necessita para desenvolver o seu trabalho.

O modelo da base de dados deverá ser baseada na figura em anexo.

Esta base de dados implica a criação de tabelas selecção do tipo de campos e tamanhos. Por outro lado deverá ser estabelecido o relacionamento entre tabelas, no estabelecimento do relacionamento deverá ser escolhido ou não as eleminações e actualizações em cascata, isto implica a análise do tipo de relacionamento.

  1. Tarefas:

-Criar consultas para obter:

    • Remates por jogador (Baliza e Falhados)

    • Passes completos por jogador por jogo.

    • Média de remates por jogo

    • Média de passes perdidos por equipa.

-Criar relatórios para obter:

    • Listagens de jogadores

    • Listagens de Jogo

-Criar os formulários:

    • Para todas as tabelas

    • Menu principal, de forma a abrir os formulários

  1. Ligação ao Winword

    2. Analisar a possibilidade da defenição de uma ligação da base de dados ao Word, de forma a ser possível a realização de uma impressão em série. Neste caso deverá ser considerada a tabela de atletas comos os destinatários de um Mailing para todos os atletas que estão na tabela.

  2. Um relatório com o máximo de 2 páginas sintetizando as dificuldades encontradas e sugestões para a melhoria da estrutura da base de dados.

  3. O trabalho deverá ser entregue já com as tabelas préviamente carregadas para ser possível analisar o trabalho.

Obs: A disquete deverá conter a Base de Dados, Documento para a impressão em série e relatório do trabalho.
