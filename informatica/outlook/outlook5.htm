Requisitos de sistema

Para utilizar o Microsoft Outlook 2000, é necessário:

  • Computador pessoal ou multimédia com um processador Pentium 75 ou superior
  • Sistema operativo Microsoft Windows 95 ou posterior, sistema operativo Microsoft Windows NT Workstation 4.0 ou posterior (com Service pack 3 ou posterior) ou posterior
  • 16 MB de RAM para o Windows 95 ou 32 MB de RAM para o Windows NT, mais 8 MB de RAM para o Outlook
  • 153 MB de espaço livre no disco rígido (este numero refere-se à instalação típica, podendo variar muito numa instalação personalizada)
  • Unidade de CD-ROM
  • Adaptador vídeo VGA ou de resolução superior (Super VGA recomendado)
  • Microsoft Mouse, Microsoft IntelliMouse™ ou dispositivo apontador compatível
  • O Acesso à Internet é necessário para utilizar as funcionalidades Internet

Requisitos adicionais ou serviços adicionais para determinados componentes:

  • Modem de 9600 baud; 14,400 ou superior (recomendado superior)
  • Computador Multimedia para ter acesso a sons ou outros efeitos multimedia

  • Para usar o correio electrónico é necessário O Microsoft Mail, Microsoft Exchange, Internet SMTP/POP3, IMAP4, ou outro sistema de messaging compatível com MAPI
  • Para usar no Outlook algumas funções avançadas de colaboração é necessário usar o Microsoft Exchange Server no servidor
  • As funções que usam a Internet requerem o pagamento a um ISP de um serviço adicional
