Data Central: The Microsoft Project Database

The data you enter in the Microsoft Project interface travels to Microsoft Project’s database. A database is nothing more than a special container for storing data in a very orderly way, as you might if you compiled a table that listed construction tasks in one column, the resources for those tasks in another, and the task cost in the last column.

In a database, a column is called a field, such as the Task Name field. Each field stores a particular kind of information.
