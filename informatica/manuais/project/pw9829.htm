Share Information with Simplified Importing and Exporting

It used to be that you practically needed a Ph.D. to import and export data between programs. You had to know all about the file formats in which each program saved its data and how to convert from one file format to another. It could be especially challenging to exchange data with a database if the field names differed from the corresponding field names in the other database.
