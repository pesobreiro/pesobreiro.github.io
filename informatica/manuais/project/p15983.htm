Format Text Elements in Sheet Views

Microsoft Project does some sheet-view formatting for you automatically; for example, it makes summary tasks bold. But if this automatic formatting doesn’t highlight the information you want in just the way you want it, you can format these elements yourself. Column headings, individual pieces of information, and even entire categories of information can easily be rewritten, realigned, and reformatted.
