How Microsoft Project Schedules a Task

Microsoft Project schedules a task based on information that you enter into your project plan (such as the task duration), the type of link you assign to the task, resource assignments, and constraints. When you vary any one of these factors, you can affect the task’s schedule. To have Microsoft Project schedule a task exactly the way you want it, you need to know how these factors do that.
