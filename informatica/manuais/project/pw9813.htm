Multiple Resource Rates

Most resources work on one kind of task for which they’re paid a rate appropriate to that task. Occasionally, though, a resource may work on several kinds of tasks, each kind commanding a different pay rate. When you assign a resource to several tasks requiring different payment schedules, you can enter different rates into the resource’s cost rate tables. Microsoft Project then applies the correct rate to each task the resource works on.
