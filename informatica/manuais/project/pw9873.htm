Connect to Microsoft Project Information on the Internet

From product discussion forums to technical documents, you’ll find a wealth of information about Microsoft Project on the Microsoft Project Home Page, located at:

http://http://www.microsoft.com/project/

The Microsoft Project Home Page includes the following useful information:

  • An overview that explains how Microsoft Project can help you manage your projects more effectively.
  • Case studies that show how organizations like yours have put Microsoft Project to work for them.
  • Articles about project management that provide valuable insight that you can apply to your own project- planning process.
  • New templates and macros that help you work more efficiently with Microsoft Project.
  • Detailed reviews that reveal why Microsoft Project has won more awards than any other project management program.
  • Links to other Microsoft products that you may want to use along with Microsoft Project, such as Microsoft Excel, Microsoft Access, and Microsoft Team Manager.

To connect to Microsoft Project information directly, point to Microsoft on the Web on the Help menu, and then click Product News.
