What’s Covered in the Online Help

Perhaps you just need a quick reminder for a procedure that’s faded from memory. Or, you want a tutorial that takes you step by step through essential project management procedures. Sometimes all you need is a brief description of an unfamiliar field. Or maybe, just maybe, you couldn’t be bothered looking through this user’s guide. (Or maybe you just "misplaced" it.) For situations like these, you can look in the online Help for instructional information.

The online Help is the documentation that’s included in the Microsoft Project program; it’s the documentation that you see on your screen when you’re running Microsoft Project.

The Microsoft Project online Help has seven ways to help you.
