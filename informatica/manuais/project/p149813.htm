Available Tables

When you want to display information in a sheet view, you can choose exactly the information you want to see by selecting and applying a suitable table. You can use predefined tables or create your own to show only the fields you want. There are two kinds of tables: those that you can apply to task views (called task tables) and those that you can apply to resource views (called resource tables).
