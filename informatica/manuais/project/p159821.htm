Copy a Microsoft Project Graphic to Another Program

By adding a Microsoft Project graphic to a document in another program, you add useful information that’s visually appealing. You can copy and paste entire Microsoft Project views or selected view information as graphics into any program that is capable of displaying graphics information as images. While you can’t copy and paste selected information as graphics from the Task PERT view, Task Form view, or Resource Form view, you can do so from any other view.

To copy a graphic from Microsoft Project to another program

  1. In Microsoft Project, select the graphic you want to copy.
  2. Click Copy Picture .
  3. To copy the selected information so that it can be printed, click For printer, and then click OK.
    To copy the selected information as a GIF image file, click To GIF image file, and then click OK.
    The default is to copy the information so that it can be displayed on the screen.
  4. To copy only specific rows of information that you’ve selected, click Selected rows.
  5. To copy only information that occurs between specific dates, click Date, and then enter the dates in the from and to boxes.
  6. Switch to the document into which you want to paste the Microsoft Project information.
  7. Paste the information using the program’s Paste command.
