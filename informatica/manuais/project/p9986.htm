Assign Different Rates to a Resource

To allow for changes in resource rates, such as pay increases or decreases, you can assign different rate values to be applied at times you specify.

To assign different rates to a resource

  1. On the View Bar, click Resource Sheet .
  2. In the Resource Name field, select the resource for which you want to enter different rates.
  3. Click Resource Information , and then click the Costs tab.
    The default cost rate table, A, is on top. If you don’t specify which of a resource’s cost rate tables to use for a task, Microsoft Project uses cost rate table A by default.
  4. In the Cost rate tables, type the effective date and the new standard, overtime, or per-use-cost rate in the corresponding fields of the first blank row.
