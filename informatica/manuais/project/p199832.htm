Send Updated Status to the Workgroup Manager at Any Time

A workgroup member can send an updated status message at any time. For example, the workgroup manager may ask workgroup members to send updated task status information at the end of every week, but might not send a TeamStatus message to prompt the workgroup members to do so.

Workgroup members can send updated status at any time by using Microsoft Outlook or the TeamInbox. In both cases, the workgroup member must first receive a TeamAssign message from the workgroup manager. Microsoft Outlook can incorporate tasks from a TeamAssign message into its task list. The task information can be updated in Microsoft Outlook and then sent to the workgroup manager.
