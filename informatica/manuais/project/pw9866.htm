Visual Map

By viewing the Visual Map, you can see at a glance the essential steps to creating and managing a project, as well as the four main phases of the project management process. The Visual Map’s numbered steps help you perform all necessary project management tasks in a logical order and keep track of tasks you’ve completed and tasks you still need to do. Clicking a numbered step displays a set of instructions for performing that step.
