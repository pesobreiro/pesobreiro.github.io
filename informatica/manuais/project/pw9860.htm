Part 5: Monitoring Your Project’s Vital Signs

Microsoft Project’s progress-tracking features give you a look-ahead capability. For instance, if a task’s schedule is starting to slip, a glance at your project plan will tell you not only how the slip will affect the schedule, but what you can do right now to prevent it. You’ll learn how to set a baseline, enter actual project data, and compare and analyze the difference between baseline and actual data.
