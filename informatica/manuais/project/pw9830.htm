Importing and Exporting Follow a Simplified Path

Regardless of the kind of program from which you import or to which you export, you follow one main path to perform the task. When you import data, you always begin with the Open command. When you export data, you always begin with the Save As command. With this unified approach, importing and exporting have gotten nearly as easy as opening and saving a project file created in Microsoft Project.
