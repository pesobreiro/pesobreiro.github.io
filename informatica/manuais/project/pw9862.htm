Appendixes

There are three appendixes in this user’s guide. The first is a list of the limits of various items you can have in Microsoft Project. The list contains information such as the maximum number of tasks you can have in a project, the maximum number of projects you can have in a consolidated project, and the maximum number of levels you can have in a task outline. The second contains several case studies that describe how six companies have benefited from using Microsoft Project. And the third is a description of the accessibility options for people with disabilities.
