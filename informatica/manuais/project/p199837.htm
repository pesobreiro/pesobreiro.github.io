Change the WebInbox Password - for the Workgroup Manager

As the workgroup manager, you receive workgroup messages in the WebInbox within Microsoft Project.

To change the WebInbox password in Microsoft Project

  1. Open your WebInbox.
  2. If you’ve set up a password for your WebInbox, enter the password when prompted.
  3. Click Password.
  4. In the Old password box, type your old password.
    If you’ve never entered a password in the WebInbox, the Old password box is unavailable.
  5. In the New password box and the Confirm new password box, type your new password.
