Create a Project Plan

During this phase you enter tasks, estimate how many days or hours it will take to complete each task, specify when one task should start relative to another, and assign resources to tasks. Microsoft Project will:

  • Schedule each task and calculate the project finish date.
  • Calculate the time required to complete each task.
  • Balance scope, resources, and time — or show you where they’re out of balance.
