Set Up a Printer or Plotter

Before you can print from Microsoft Project, you must set up your printer or plotter, if you haven’t already done so. You can print on any printer or plotter supported by your system software.

In Microsoft Windows 95 and Microsoft Windows NT, you set up your printer for use with all your programs. You can switch to a different installed printer or change printer settings as you work in Microsoft Project.

To select a printer or plotter and change its settings in Microsoft Project

  1. On the File menu, click Print.
  2. In the Name box, click a printer or plotter, and then click Properties.
  3. Select the options you want to use, and then click OK.
  4. Click Close.
