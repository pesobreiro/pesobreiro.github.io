Microsoft Project Help Topics

When you’re working with Microsoft Project and need an answer, a solution, or some background information "on the fly," the place to turn is the comprehensive set of online Help topics. The Help topics consist of step-by-step instructions for basic as well as advanced project management procedures, conceptual topics that explain the goals and consequences of procedures, and illustrated examples that help you apply the procedures you learn to your own work.

To view Help topics, use the commands on the Help menu. If you click Microsoft Project Help, you can type a question to the Office Assistant, which responds by presenting you with a list of 5 topic titles (you can expand the list to 10), from which you choose the topic that best answers your question. If you click Contents and Index, you display the traditional Help window that enables you to search for a topic through a booklike table of contents or an index.
