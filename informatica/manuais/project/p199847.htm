View and Forward a Routed Project File

After you’ve completed the cycle of receiving a project file attached to an e-mail message, reviewing the schedule, making changes, and adding comments, you can return the project file to the originator of the routed message or forward the message to the next person on the routing list.

To forward a routed project file

  1. In your e-mail program, open the routed message.
  2. To open the project, double-click the Microsoft Project icon in the e-mail message.
  3. Review and modify the project, and then click Save .
  4. On the File menu, point to Send To, and then click Next Routing Recipient.
  5. To send the project to the next recipient on the routing list, click OK.
