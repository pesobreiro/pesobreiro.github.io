Microsoft Project 101: Fundamentals

Microsoft Project 101: Fundamentals gives you richly illustrated answers to six common questions about how you and Microsoft Project work together to create and refine a project schedule. These questions are:

  • How do I manage projects with Microsoft Project?
  • What project information do I enter?
  • Where do I enter project information?
  • How does Microsoft Project create my schedule?
  • How do I display project information?
  • How do I fine-tune my project plan?

No procedures here, but in fewer than 15 minutes the concise text and detailed illustrations provide you with the background information you need to create an effective project plan and work with Microsoft Project efficiently.
