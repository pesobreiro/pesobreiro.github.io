Getting Assistance While You Work

Few people have a lot of time to learn new software, so we’ve made it easy to learn Microsoft Project while you use it. The Microsoft Project User’s Guide provides step-by-step instructions and in-depth background information for the procedures you’re most likely to perform. Using online Help, you can quickly find just what you want to know, without searching through a lot of unnecessary information.
