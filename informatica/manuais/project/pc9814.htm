Which Files to Download or Order

You can obtain these accessibility files by downloading them with your modem or by ordering them on disks by phone. Specific information about downloading or ordering these files immediately follows this list of files.

The accessibility files include:

  • Application notes that provide more complete documentation on ways to customize Windows and Windows NT.
  • Access Packs and Dvorak keyboard layouts that provide additional features for versions of Windows or Windows NT in which they are not already included.
