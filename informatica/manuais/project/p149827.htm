You Can Create or Modify a View, Table, or Filter

Microsoft Project comes with many predefined views, tables, and filters, yet on occasion you might not find the one that displays exactly the information you want in just the format you want. In that case, you can create or modify a view, table, or filter. You can create a view, table, or filter from scratch, create one based on an existing one, or edit an existing one. When you save your project, your new or modified view, table, or filter is saved with the project. You can use a new or modified view, table, or filter only in the project file in which you created or modified it, unless you make it available to other project files by using the Organizer. For information about using the Organizer, see the online Help.
