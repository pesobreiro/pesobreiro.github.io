A Task Usage View with Resource Details

The Task Usage view is a list of tasks showing assigned resources grouped under each task. (This is the reverse of what the Resource Usage view shows.) You can use the Task Usage view to assign people and other resources to tasks, enter and edit task and resource information together, distribute task assignments more evenly across resources, find out how many hours each resource is scheduled to work on particular tasks, and vary the amount of work a person spends on a task by setting work contours.
