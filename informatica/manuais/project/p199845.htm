Send a Project File

To inform people in your workgroup about the project schedule, you can give each member a printed copy to review, and then call a meeting to get comments or else speak to each member individually. An easier and more efficient way to inform workgroup members and get their feedback in one stroke is to send a single e-mail message that contains the entire project file. To review the schedule, recipients must have Microsoft Project.

To send an entire project file

  1. Open the project file.
  2. On the File menu, point to Send To, and then click Mail Recipient.
  3. In the To box, type the e-mail names of the recipients.
  4. In the message area, type your message.
  5. Click Send.
