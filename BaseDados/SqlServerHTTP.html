Exposição de uma BD via HTTP

Para realizar a exposição de uma base de dados pode utilizar os seguintes tipos de URL's que podem ser acedidos através de Virtual Names:

  • Template - Utilização de um ou mais comandos SQL num documento XML
  • XPATH - Executar XPATH Queries que são convertidas em comandos XML
  • dbobjects - Aceder a objectos da base de dados (Tabelas, Views, Sprocs, ...)
  • WSDL - Utiliza SOAP para enviar pedidos ao SQLXML acendendo a um ficheiro WSDL
  • SOAP - Suporta o envio de HTTP SOAP
