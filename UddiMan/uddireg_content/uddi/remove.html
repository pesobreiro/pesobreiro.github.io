Removing the UDDI Registry application from WebSphere

The UDDI Registry application can be removed from the WebSphere Application Server as follows:

  1. Start the WebSphere Administrator's Console (if it is not already running).
  2. Expand the administrative domain in the left hand pane so that you can see the 'UDDI Registry' application under 'Enterprise Applications'.
  3. Make sure the application is stopped, either by right-clicking it and selecting Stop, or by selecting it and clicking on the Stop symbol on the actions bar of the Administrative Console.
  4. Remove the UDDI Registry application by right-clicking it and selecting Remove (or by selecting it and clicking on the Remove symbol).
  5. Reply 'No' when asked if you want to export the binding information, then reply 'Yes' to confirm you want to remove the UDDI Registry and the objects it contains.
  6. Finally click 'OK' when the remove completes.

These instructions are for WebSphere Application Server Advanced Edition (AE). If you are using WebSphere Application Server single-server edition for Developers (AEd), please refer to the section on "Uninstalling applications with the Web console" in the InfoCenter for AEd. Please note that for AEd, you need to also delete the directory corresponding to the application from the installedApps directory of the WebSphere AppServer.
