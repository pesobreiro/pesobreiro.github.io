Applying service to the IBM WebSphere UDDI Registry

Service for the IBM WebSphere UDDI Registry will normally be shipped as an updated version of the UDDI Registry application file, uddi.ear. To apply the updated file, you should do the following:

  1. Remove the UDDI Registry application from the WebSphere Application Server, by following the instructions in "Removing the UDDI Registry application from WebSphere".
  2. Save a copy of the previous uddi.ear file, which is located in the installableApps subdirectory of the WebSphere install location, in case of any problems with the new file, or if you are testing it and wish to reinstall the previous version when you have finished testing.
  3. Copy the updated uddi.ear file into the installableApps subdirectory of the WebSphere install location.
  4. Install the UDDI Registry application into the WebSphere Application Server, by following the instructions in "Installing the UDDI Registry application into WebSphere".
