Terminology

Throughout this Information Center reference is made to the location for installation of the IBM WebSphere product as <WebSphere-install-dir>. This translates, by default, to the following locations:

Windows

C:\WebSphere\AppServer\

Linux

/opt/WebSphere/AppServer/

Similarly reference is also made to the location for the IBM WebSphere UDDI Registry as <UDDI-install-dir>. This translates, by default, to the following locations:

Windows

C:\WebSphere\UDDIReg\

Linux

/opt/WebSphere/UDDIReg/
