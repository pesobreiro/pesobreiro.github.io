Welcome to the IBM WebSphere UDDI Registry InfoCenter©

The IBM WebSphere UDDI Registry InfoCenter provides fast, centralized access to product information. Use the table of contents on the left to view the various topics for a specific product or technology. Select the topic you are interested in to either open documentation locally or find information about how to locate documentation.

The banner also provides other ways to navigate through the information.

  • Contents loads an applet-based table of contents that allows you to show and hide sections of the table of contents. If the applet does not load, use the HTML link to access an HTML version of the table of contents.
  • Search launches the search feature, if available. When you click Search a new window is displayed with an input field and Search button.
  • HTML loads an HTML table of contents. The HTML version allows you to view the table of contents if the navigation applet does not load.

The WebSphere InfoCenter uses HTML 4.0, Cascading Style Sheets (CSS), JavaScript, and Java applets to provide advanced navigation features for a variety of computing platforms, such as Windows and Unix. For more information, see Using the InfoCenter.
