Gestão dos Stocks

O Serviço Informático, na pessoa do Dr. Jorge Madeira, é responsável pela gestão dos seguintes artigos :

  • CD virgens
  • Toners para impressoras
  • Disquetes
  • Tinteiros para impressoras
  • Tinteiros para Faxes

Cada artigo têm um código atribuído e partir daí é realizado um movimento de stock (entrada/saída) para um departamento, centro ou serviço. Sendo registado a data do movimento, o responsável pelo o pedido e a quantidade.

Os consumiveis pretendidos deverão ser levantados no Serviço Informático.
