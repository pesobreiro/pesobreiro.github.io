The Developer Solutions Sample Application

The Developer Solutions sample application contains examples of the forms, reports, and interface elements you most commonly use to create applications.

   To open the Developer Solutions sample application

1  &nbspStart Microsoft Access.

2  &nbspIn the Microsoft Access dialog box, make sure that Open an Existing Database is selected, and then click OK.

3  &nbspIn the Open dialog box, navigate to the Office folder in the Look In box, double-click the Samples folder, and then double-click Solutions.

Developer Solutions displays a startup form, and then automatically opens the Developer Solutions form. The Developer Solutions form lists all the examples in the sample application by category.

Tip   When you open a form or report in Developer Solutions, the Show Me topic for that form or report is automatically displayed if the check box at the bottom of the Developer Solutions form is selected.

Note   If you close the Show Me window, or if you clear the check box on the Developer Solutions form that automatically displays Show Me topics, you can display a topic by either clicking the Show Me button on a form's toolbar, or searching the Help index for the name of the topic.

A Sample of What You'll Find in Developer Solutions

The sample forms and reports in Developer Solutions cover the main areas of application design, from changing a form's toolbars in response to events, to creating a crosstab report with dynamic column headings.

Sample Forms

Forms are the major component of a Microsoft Access application. You use them not only to enter and view data, but also as a way to organize your application into a coherent system. In Developer Solutions, there are examples of forms that:

&nbspSwitch between entering and editing records.

&nbspFill a record with data from the previous record.

&nbspSearch for records based on values you enter in one or more fields.

&nbspProgressively display more detail about a given subject.

&nbspSynchronize data between two forms.

&nbspCollect and store temporary information.

&nbspSelect more than one item in a list box.

In many cases, several techniques are combined on one form, as shown in the EnterOrEditProducts form.

To display this form, in the Developer Solutions form, click Sample Forms in the Select A Category Of Examples box, and then click EnterOrEditProducts in the Select An Example box.

Sample Reports

With most applications, you'll want a way to format and print data for distribution. The reports in Developer Solutions show you how to:

&nbspCreate a report for a preprinted form.

&nbspCreate a form letter.

&nbspCreate crosstab reports.

&nbspDraw a circle around important information.

For example, the SalesLetter report uses Visual Basic code to draw a circle around the sales totals that meet or exceed the goal. To preview this report, in the Developer Solutions form, click Sample Reports in the Select A Category Of Examples box, and then click SalesLetter in the Select An Example box.

Sample Code

Developer Solutions also includes procedures that you may find useful when developing your own applications. Procedures included in Developer Solutions show you how to:

&nbspLink tables at startup.

&nbspClose all objects when the user exits the application.

&nbspCapitalize the first letter of every word in a field.
