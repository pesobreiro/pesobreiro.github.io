  • To create a boot usb using a ISO:
  • execute diskutil list
  • sudo umount /dev/disk1s1 replace disk1s1 by the USB identifier
  • sudo dd if=~/Desktop/Microwatt-R10-32bit.iso of=/dev/rdisk1s1 bs=1m
  • diskutil eject /dev/disk1s1
    • Carefull if the spotlight is working in the USB, it should be disabled to avoid the message that its busy when is ejected. Another option is to use the GUI https://github.com/thefanclub/dd-utility or try the UNetbootin
  • Shared folder beetween virtualbox running linux and host: (1) Install Guest Additions; (2) Define shared folder, e.g. name Dropbox path c:\Dropbox; (3) create folder "Share" in the home directory; (4) sudo mount -t vboxsf Dropbox ~/Share/

  • bootrec /fixmbr
  • bootrec /fixboot
  • bootrec /scanos
  • bootrec /rebuildbcd > Type A
    Command line tools
  • head and tail: utilities for showing a specified number of lines from the top or bottom of the file. Default 10. Let's try it with our file. tail -n 3 csvfile.csv
  • wc: Word count. count the number of bytes, characters, words, or lines in the file. wc -l file.csv
  • With grep, you can search a file or other input for a particular pattern of characters. grep -i "text" file.csv
  • tr: translate a file from a format to another. grep "20 Jan 2017" file.csv | tr ',' '\t' > file.tsv. Translate file to transform comma by tabs to tab delimited file.
  • awk: AWK is an excellent tool for processing rows and columns. awk -F "\t" '{print $3 " " $NF}' jan20only.tsv
    The -F "\t" we pass to awk simply tells it that we're working with tab-separated data. Within the braces, we're actually telling awk to execute just a little bit of code. We're telling it to print the third column with $3, and then the last column with $NF (the "number of fields"), and place two spaces between them to make it a little more legible
