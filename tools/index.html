Tools

  • A good cloning util Clonezilla, more options are available here.
  • A great text editor Sublime
  • Simple tool to recover data Photorec and testdisk
  • How to clone several computers here.
  • Rsync a good tool for backup. More information about configuring and instaling rsync for windows backup is available here. Use DeltaCopy client on source and rsync on target. You can either run rsync as a daemon or use it via SSH.Refer to rsync documentation for implementation details.
  • To create a boot usb using a ISO:
  • execute diskutil list
  • sudo umount /dev/disk1s1 replace disk1s1 by the USB identifier
  • sudo dd if=~/Desktop/Microwatt-R10-32bit.iso of=/dev/rdisk1s1 bs=1m
  • diskutil eject /dev/disk1s1
    • Carefull if the spotlight is working in the USB, it should be disabled to avoid the message that its busy when is ejected. Another option is to use the GUI https://github.com/thefanclub/dd-utility or try the UNetbootin
  • Shared folder beetween virtualbox running linux and host: (1) Install Guest Additions; (2) Define shared folder, e.g. name Dropbox path c:\Dropbox; (3) create folder "Share" in the home directory; (4) sudo mount -t vboxsf Dropbox ~/Share/
  • Recover windows 10 boot:
  • ALT+F10 or F8 to access the windows recovery
  • Repair computer > Troubleshoot > Advanced options > Command prompt
