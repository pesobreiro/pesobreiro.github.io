How to use diferent versions of python without messing arround already existing environment

  • Check existing versions: conda search python
  • Create a new environment: conda create --n environmentName python=3
  • Activate a new environment: Windows=activate environmentName and Linux = source activate environmentName
  • Create Python 3.5 environment: conda create -n py35 python=3.5 anaconda
  • Create Python 2.7 environment: conda create -n py27 python=2.7 anaconda
  • Update Python in the environment: conda update python
