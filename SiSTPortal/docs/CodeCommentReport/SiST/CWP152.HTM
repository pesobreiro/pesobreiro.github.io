SiST.UddiAPI.UddiData.URLType Enum
Enumeração dos diversos tipos de URLs que são possíveis utilizar no accessPoint

Access: Public
  Members Description  
    mailto indica que o accessPoint está formatado como um endereço que correio electrónico ex. mailto: user@subdominio.dominio

 
    http http: o accessPoint contêm o URL com o endereço HTTP ex. http://subdominio.dominio/local

 
    https http: o accessPoint contêm o URL com o endereço HTTPS ex. https://subdominio.dominio/local

 
    ftp ftp: o accessPoint contêm o endereço FTP ex. ftp://subdominio.dominio/local

 
    fax fax: o accessPoint contêm um número de telefone de uma máquina de fax ex. 123 123 123123123

 
    phone phone: o accessPoint contêm um número de telefone ligado a uma pessoa ou um equipamento de resposta automática (ex. 123 123 123123123).

 
    other other: outro tipo de endereço
