Classification

There are several algorithms available. This type of algoritms deal with classification problems where the outputs uses only discrete values aka categorical variables

The goal of a classification is to find a functional mapping between the input data X and a class label Y (e.g. -1 or +1), such Y=f(X). The mapping is created using training data supplied to the classification algorithm where the goal is to predict correctly unseen data.

There are several algorithms that can be employed. The Support Vector Machine are interesting to use because their capability to work with small sets of data e.g. 50.

The performance consider the following elements:

  • Confusion Matrix:
    • True Positives (TP): True positives are the cases when the actual class of the data point was 1(True) and the predicted is also 1(True), e.g. positive well-being with predicted positive well-being or predicting churn and in the actual values there is a churn. Predict positive well-being and the true value is positive.
    • True Negatives: True negatives are the cases when the actual class of the data point was 0(False) and the predicted is also 0 (False), e.g. negative well-being with predicted negative well-being or predicting that a customer dropout and he droped.
    • False Positives: False positives are the cases when the actual class of the data point was 0(False) and the predicted is 1(True), e.g. predicted positive well-being with negative well-being or predict a customer dropout and the customer is retained.
    • False Negative: Predicted wrongly FALSE and the predicted value is 0 when in reality is 1, e.g. predict well-being negative when is positive.
