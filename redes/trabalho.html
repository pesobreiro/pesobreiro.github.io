Trabalho

O trabalho a ser desenvolvido baseia-se na elaboração de um relatório que aborda a instalação e configuração de servidores, que será realizado por cada um dos grupos de trabalho.
Os temas de trabalho são sobre:

  • Políticas para Active Directory
  • Instalação remota e response files
  • Internet Security and Acceleration Server (ISA) 2000
  • Exchange Server
  • Instalar um servidor em Linux que funcione como PDC (Windows NT) numa rede com utilizadores 2000

Políticas para Active Directory

Implementar um servidor que permita que controle e realize a gestão de uma rede dos utilizadores, que deverão ter as seguintes restrições:

  1. Só têm acesso ao Word, Excel e ao Outlook express
  2. Têm definido como imagem do ambiente de trabalho o logotipo da empresa, bem como um screensaver apropriado.
  3. Não podem alterar as definições do ambiente de trabalho
  4. Não têm permissões para instalar programas
  5. Não têm acesso ao painel de controle
  6. Não têm acesso à linha de comandos
  7. Não podem executar aplicações a partir do menu iniciar

Deverão ser implementadas medidas que permita a utilização de Roaming profile, suportando o acesso do utilizador às suas definições pessoais mesmo que seja realizado a substituição por motivos de avaria.

Por último, deverão estar implementadas medidas de segurança para os users que permita o acesso só à sua área de trabalho e realize os backups dos dados periódicamente do servidor.

Instalação remota e response files

Requisitos para instalação remota:

  1. Implementar um servidor RIS
  2. Criar as imagens com o RIPPrep

Answer files:

  1. Criar answer file que permita uma instalação por parte do técnico com menos interacção possível
  2. Garantir a resposta às questões colocadas durante o setup

Internet Security and Acceleration Server

Instalar um servidor com o ISA server onde serão adicionadas algumas restrições, nomeadamente:

  • Cortar o acesso a alguns sites onde se apresenta uma página alternativa a convidar o utilizador a aceder a outros conteúdos, sugerindo inclusivé alguns sites.
  • Só permitir o acesso através dos protocolos http, smtp e pop aos utilizadores durante o período normal de trabalho.
  • Permitir aos utilizadores durante a hora de almoço a utilização sem restrições

O servidor ISA deverá ser preparado para dois tipos de utilizadores os users onde se aplicam as restrições referidas anteriormente e os administrators, que não têm qualquer restrição.

Exchange server

O Exchange server deverá ser implementado com duas contas e testado com envio e recepção de mensagens entre os dois utilizadores.

Com as seguintes restrições:

  • A caixa do correio está limitada a 5Mb
  • Não é permitido o Relay no servidor
  • Os utilizadores podem enviar e receber mensagens de qualquer destinatário

Não se devem esquecer de implementar o connector e salvaguardar as devidas restrições.

Servidor Samba

Quem quiser ser diferente, pode implementar um servidor linux com o samba que implemente as funcionalidades de um PDC, falar com o professor para receber documentação de apoio.
