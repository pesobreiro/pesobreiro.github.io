Como Instalar o Bind

Pressupostos:
O nome do Dominio: redes.pt
Ip de identificacao da rede: 195.60.168.0
Mascara: 255.255.255.128

Servidor de dns: 195.60.168.50 (subdominio:ns1)
Servidor de www: 195.60.168.50 (subdominio:www)

    Passos a seguir para a instalação do Bind:
    1. 1º Definir um ficheiro com o nome named.conf, que contem a informação para os ficheiros que vão ser utilizados pelo servidor de DNS.
    2. 2º Criar o ficheiro da zona(zone file) com o nome de "redes.dns" que ira conter as definicoes do dominio que estamos a configurar.
    3. 3º Criar o ficheiro para o reverse da zona com o nome de "0.168.60.195.in-addr.arpa.dns" que contem as definicoes a resolucao IP->Nome
    4. 4º Indicacao para resolucao por parte dos clientes do servidor de dns, que e definido no ficheiro "resolv.conf"
Named.Conf

# informacao relativamente a zona
zone "redes.pt" {
type master;
file "redes.dns";
};

zone "168.60.195.in-addr.arpa" in {
type master;
file "168.60.195.in-addr.arpa.dns";
};

redes.dns

@ SOA ns1.redes.pt. root.ns1.redes.pt. (
000001 ; serial number
36000 ; refresh em 10 horas
3600 ; retry em 1 hora
3600000 ; expire em 42 dias
360000 ; duracao minima de 100 horas
)
; Informacao relativamente ao servidor de DNS
IN NS ns1.redes.pt.
IN NS www.redes.pt.

IN A 195.60.168.50
IN MX 100 mail

localhost IN A 127.0.0.1

; subs no redes

ns1 IN A 195.60.168.50
www IN CNAME ns1
mail IN A 195.60.168.51

resolv.conf

search redes.pt
domain redes.pt
# Redes
nameserver 195.60.168.50
