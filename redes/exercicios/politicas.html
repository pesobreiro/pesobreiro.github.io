Políticas para utilizadores

Pretende-se controlar numa empresa os utilizadores, para isso deverão ser definidos dois tipos de utilizadores, os super users e os users. Os super users tem a possibilidade de alterar as definições do computador, têm acesso a todos os programas e não podem instalar novos programas, os users têm limitado a configuração e instalação de programas bem como no acesso às aplicações. Os users têm as seguintes limitações:

  • Só têm acesso ao wordpad e bloco de notas.
  • Não podem alterar as definições do ambiente de trabalho
  • Não têm permissões para instalar programas
  • Não têm acesso ao painel de controle
  • Não têm acesso à linha de comandos
  • Não podem executar aplicações a partir do menu iniciar

Os testes deverão realizados com o computador a correr o windows 2000 server e windows 2000 profissional.

Os users deverão estar definidos numa Organizational Unit(OU) users.
