Task Scheduler

Pretende-se criar uma tarefa que arranque a uma determinada hora e que proceda ao backup dos dados do utilizador para um outro computador(área de trabalho no servidor) .

Na realização do backup deverão ser considerados os seguintes elementos:

  • Documentos do utilizador.
  • Emails
  • Favoritos
  • Elementos que estejam no ambiente de trabalho

Para testar os backups, deverá ser realizado o teste num computador da mesma rede, onde esteja definido uma partilha que permita as cópias dos dados.
